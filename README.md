# Trippple

## Installation

### Pre-requisites

* Yarn
* Watchman
* xCode (for iOS simulator)
* Android SDK (for Android simulator)

### Installation

```
yarn
```

## Getting started

### Web

```
cd ./packages/web
yarn start
```

### App

```
cd ./packages/app
yarn start          # Compiler / file watcher
yarn run ios        # iOS simulator
yarn run android    # Android simulator
```
